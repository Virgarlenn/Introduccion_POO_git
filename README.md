# Introducción a POO y Git 🐍

Repositorio de la clase de **Programación Orientada a Objetos** y **Git/GitHub**.

## 🎯 Objetivos de la clase

- Entender qué es una clase, un objeto, un atributo y un método.
- Poder crear nuestras primeras clases en Python.
- Aprender el flujo básico de trabajo con Git: `clone`, `status`, `add`, `commit`, `push`.
- Subir su primer ejercicio resuelto al repositorio del grupo.

## 🚀 Comandos útiles para esta clase

| Comando | ¿Para qué sirve? |
|---|---|
| `git clone <url>` | Descargar el repo a tu computadora por primera vez |
| `git status` | Ver qué archivos modificaste |
| `git add <archivo>` | Marcar un archivo para subir (staging) |
| `git commit -m "mensaje"` | Guardar los cambios marcados con una descripción |
| `git push` | Subir los cambios a GitHub |
| `git pull` | Traer los cambios que subieron los demás |

Para más detalle de cada comando, vayan a [`teoria/02-guia-git.md`](teoria/02-guia-git.md).

## 📝 ¿Cómo entrego el ejercicio?

1. Cloná el repo (o hacé `pull` si ya lo tenés).
2. Completá el código en `ejercicios/01-mi-primer-objeto/`.
3. `git add .`
4. `git commit -m "Resuelvo ejercicio 1 - Mi primer objeto"`
5. `git push`

¡Listo! Con eso ya quedó subido para que lo revise.
